:: Registro de información e Históricos :: para la definición y gestión del historial generado por los componentes respectivos conectados al bus de integración. En este módulo, el usuario podrá llevar a cabo las siguientes operaciones:

  • Definir nuevos roles para usuarios: Los roles son utilizados para dar limitación de acceso al la aplicación de históricos, de esta forma un determindao usaurio puede solamente consultar, mientras que uno con más permisos podrá escribir históricos a demas de consultarlos o incluso crear más usuarios o roles.
