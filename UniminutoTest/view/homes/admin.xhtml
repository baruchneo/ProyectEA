:: Recepción de información :: para la definición y gestión de las entidades asociadas al módulo de recepción de archivos. En este módulo, el usuario podrá llevar a cabo las siguientes operaciones:

  • Definir nuevos procesos de negocio: un proceso de negocio representa un ambiente independiente de recepción de información. Para este ambiente se define la obligatoriedad del envío con firma digital, los directorios de procesamiento de información y el indicador de proceso en mantenimiento.


  • Definir tipos de informes: un proceso de negocio debe tener asociado por lo menos un tipo de informe. En un tipo de informe se define la periodicidad de envío, la nomenclatura de renombramiento de archivos y los tipos de entidades que deben transmitir la información.


  • Gestionar entidades: el usuario podrá seleccionar una entidad determinada y actualizar sus indicadores de envío de información con firma digital o archivos de notas anexas.


  • Auditoría: el usuario podrá seleccionar los envíos realizados por la entidades vigiladas, ver su estado de procesamiento y autorizar reenvíos de información.
