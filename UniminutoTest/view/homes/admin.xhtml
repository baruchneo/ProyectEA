:: Registro de información e Históricos :: para la definición y gestión del historial generado por los componentes respectivos conectados al bus de integración. En este módulo, el usuario podrá llevar a cabo las siguientes operaciones:

  • Definir nuevos roles para usuarios: Los roles son utilizados para dar limitación de acceso al la aplicación de históricos, de esta forma un determindao usaurio puede solamente consultar, mientras que uno con más permisos podrá escribir históricos a demas de consultarlos o incluso crear más usuarios o roles.


  • Definir usuarios: Los usuarios se definen de aquel que cuenta con un identificador y una clave de acceso para hacer uso del servicio, esto le permite acceder a información no disponible para aquellas personas que no están registradas en el sistema.


  • Definir los Centros Medicos: Los centros medicos estan destinados a la atencion medica de la poblacion. Prestan un servicio de urgencias para poder atender las emergencias que se produzcan.


  • Definir Alerta: Un estado o una señal de alerta es un aviso para que se tomen las precauciones necesarias, hay varios tipos de alerta que se emiten por el sistema para saber el tipo de riesgo del paciente, monitoriando un seguimiento de los signos vitales.


  • Definir Rango: Los rangos son niveles que indican la medida de una cantidad en relación a una escala especifica .
